# Регрессионный анализ набора данных Housing
Этот репозиторий содержит код множественного регрессионного анализа с целью объяснения какой из факторов оказывает наибольшее влияние на стоимость дома в округах Калифорнии
## Данные
Используется набор данных Housing в 20640 наблюдений. Для анализа была составлена стратифицированная выборка (2000 наблюдений) \
Целевая переменная — это средняя стоимость дома для округов Калифорнии (medianHouseValue), независимые переменные (факторы влияния): longitude, latitude, housing_median_age, total_rooms, total_bedrooms, population, households, median_income.\
Информация об атрибутах: \
•	Longitude (долгота): показатель того, насколько далеко на запад находится дом; чем больше значение, тем дальше на запад \
•	Latitude (широта): показатель того, насколько далеко на север находится дом; чем больше значение, тем дальше на север \
•	housing_median_age (медианный возраст дома): медианный возраст дома в квартале; чем меньше число, тем новее здание.\
•	total_rooms (общее количество комнат): Общее количество комнат в квартале \
•	total_bedrooms (общее количество спален): Общее количество спален в квартале \
•	population (население): Общее количество людей, проживающих в пределах квартала \
•	households (домохозяйства): общее количество домохозяйств, группы людей, проживающих в пределах жилого блока, для квартала \
•	median_income (медианный доход): медианный доход домохозяйств в пределах жилого блока (измеряется в десятках тысяч долларов США) \
•	median_house_value (медианная стоимость дома): медианная стоимость жилья для домохозяйств в пределах квартала (измеряется в долларах США) \
•	ocean_proximity (близость к океану): Расположение дома относительно океана/моря \
